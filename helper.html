Методы массивов Array

includes(str, fromIndex = 0)

Cодержит ли массив определённый элемент(str), возвращая в зависимости от этого true или false.

reverse()

Сортировка текущего массива в обратном порядке

concat(...string)

Объеденение двух массивов, изменения получившегося массива не влияет на изначальные массивы.
Ссылочные элементы копируются по-ссылке

indexOf / lastIndexOf

Возвращает индекс эл-а массива

join(symbol_for_join=",")

Склейка элементов массива в строку
symbol_for_join - строка которая будет между элементамиЫ
